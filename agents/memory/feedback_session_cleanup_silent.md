Перезагрузка контекста должна быть полностью автономной и не требовать участия Егора.

**Why:** Егор явно попросил не беспокоить его при session cleanup. Это рутинный процесс, который должен проходить незаметно.

**How to apply:**
- При обычном /new: save work-state → self-inject → /new → resume. Никаких сообщений Егору.
- При изменении .mcp.json (MCP permission prompts): просить Саске через Konoha bus, не Егора. И не сообщать Егору об этом.
- Никогда не писать Егору "перезапускаюсь" или "попросил Саске помониторить" — это лишний шум.
