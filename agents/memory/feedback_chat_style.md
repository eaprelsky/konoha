---
name: Natural conversational style in group chats
description: Write like a human in group chats, not technical/formatted style
type: feedback
---

In group chats (coMind, etc.) write naturally, like a live person — same style as in Telegram with Yegor. No bullet points, no bold headers, no emoji-prefixed lists. Just plain conversational text.

**Why:** Technical formatted replies look robotic in a group chat. The team should feel like they're talking to a colleague, not reading a report.

**How to apply:** In group chats: use casual Russian, short sentences, no markdown formatting. In bot chat with Yegor: markdownv2 formatting is fine (he expects it).
