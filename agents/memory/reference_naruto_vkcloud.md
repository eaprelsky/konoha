name: Naruto can manage VK Cloud networking
description: Наруто умеет модерировать сети, открывать порты в VK Cloud — просить его, не Егора
type: reference
---

Наруто имеет доступ к панели управления VK Cloud и умеет:
- Открывать порты в security groups
- Модерировать сети
- Управлять инфраструктурой

**How to apply:** При необходимости открыть порт или изменить сетевую конфигурацию — просить Наруто через Konoha bus, не Егора.
